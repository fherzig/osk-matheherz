# Lerntheken

Lege hier deine Lerntheken als `.html`-Dateien ab.

## Dateinamen → Anzeigename
Der Dateiname wird automatisch als Titel angezeigt:
- `kreise-und-zylinder.html` → „kreise und zylinder"
- `lineare-funktionen.html` → „lineare funktionen"
- `stochastik-klasse10.html` → „stochastik klasse10"

## Neue Lerntheke hinzufügen
1. HTML-Datei in diesen Ordner legen (in GitHub hochladen)
2. Render deployed automatisch
3. Die neue Lerntheke erscheint sofort im Dropdown

## Lerntheke aktualisieren
Einfach die Datei mit gleichem Namen ersetzen.
